# Product Overview

Kiroween is a Halloween-themed VSCode extension that provides a complete dark theme experience for the Kiro IDE. Created for the Kiroween Hackathon, it transforms the editor with spooky, vibrant colors inspired by Halloween aesthetics.

## Core Features

- Dark color theme with Halloween-inspired palette
- Custom UI theming for editor, sidebar, activity bar, status bar, and panels
- Syntax highlighting optimized for readability with thematic colors
- High contrast design (hc-black base)

## Color Palette

The theme uses a carefully selected Halloween color scheme:
- pumpkinOrange (#FF7A18) - Primary accent, keywords, status bar
- witchPurple (#6A0D83) - Sidebar, line numbers
- toxicGreen (#4EF17B) - Strings, selections
- bloodRed (#FF2E4C) - Errors, active borders, badges
- fogGrey (#C6C6C6) - Foreground text, comments
- midnightBlack (#0B0B0F) - Editor background
- voidNavy (#1A1A29) - Panels, active tabs
- hauntedBlue (#3849FF) - Functions, types, links
