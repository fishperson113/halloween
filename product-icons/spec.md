Theme: Kiroween Halloween

Style Rules:
- Icons must be monochrome SVG (single color).
- Base stroke thickness: 2px.
- No fills, only outlines.
- Glow effect handled by theme color, not inside SVG.

Palette:
- pumpkinOrange (#FF7A18) for active icons
- fogGrey (#C6C6C6) for default icons

Icon Concepts:
- Explorer = Pumpkin
- Search = Eyeball
- Git = Dead tree branch
- Debug = Lightning bolt
- Extensions = Witch hat
- Settings = Cauldron
- Terminal = Coffin
- Problems = Skull

Output Requirements:
- Each icon must be produced in pure SVG path format.
- All icons must be centered in a 24x24 viewBox.
- Kiro must output the product icon JSON manifest mapping.

Deliverables:
1. SVG asset for each icon.
2. product-icon-theme.json in VS Code schema.
